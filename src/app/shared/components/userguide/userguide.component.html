User Guide

Search, Zoom
Search and Zoom Capabilities

Navigating the MapThe scroll wheel on the mouse can be used to zoom, and in the upper left area of the tool are "+" and "-" icons to zoom in and out, respectively. The zoom level increases with each zoom in, and decreases with each zoom out. The scale bar resizes with each zoom.

Define an Area to Zoom toIn most browsers, the user can hold down the shift key, then click and drag the map to define the area to zoom in.

Home LocationThe home button will center the US on the map.

Opening a Site Information BoxTo open up a site information box, zoom in and hover over a site until the mouse hand is pointing at it, then click on the circle at its location on the map. This will open a pop-up window containing the site and project name. It also opens the Project List (on the left) and the Project Information pane (below the map).

Viewing Overlapping SitesWhen two or more sites are overlapping, their circles on the map appear darker. Clicking on overlapping sites will display a warning in the site information pop-up box.

Zoom into Overlapping SitesIn order to select overlapping sites, first zoom into them. When you are close enough to select them, a circle with a number inside will appear, representing the number of sites in that location.

Selecting Overlapping SitesOnce the circle containing the number of sites in the location is in view, click on that circle. The sites will separate, and the user can then click on each site to view their information.

Explanation

Explanation displayThe Explanation box can be expanded or collapsed by clicking on the header.

Using the ExplanationThe expanded Explanation box contains a legend for the available sites and additional layers.

 Map Selection Options
Map Selection Options

Changing the Basemap StyleThe "Basemaps" panel allows a user to show various geographic information or landscape imagery as a background layer.

Changing or Adding LayersThe "Layers" sidebar allows a user to add or remove layers of additional information.

Exploring SiGL Projects and Sites
Selection Capabilities and Site Information

The Project ListUpon initial load, the Project List contains all SiGL projects, even those without sites. The site count to the right of each project name displays how many sites each project contains.

Highlighting Project SitesClicking on a project title will apply an orange highlight to all the sites in that project, as well as open the Project Information pane.

View Site ListClick the small triangle to the left of the project name to expand the site list.

Explore Project InfoUse the Project Information pane to explore detailed information about a project and its sites. Drag the top of the box to expand, and move between the tabs to view different project and site information.

View Site ParametersFrom the "Site Information" tab, toggle between "Site Details" and Parameters Sampled" to view additional sampling information.

 Filtering SiGL Projects and Sites
Filtering SiGL Projects and Sites

Filtering Site and Project SelectionThe "Filters" sidebar allows a user to select sites or projects that match certain criteria.

Filter SitesOnce the user has selected "Change filters" on the "Filters" sidebar, the "Filter Sites" portion allows the user to select criteria to filter the site selection.

Filter Projects Once the user has selected "Change filters" on the "Filters" sidebar, the "Filter Projects" portion allows the user to select criteria to filter the project selection. These filters may select projects without any sites.

Search by Project NameAlternatively, within the "Change filters" box, the user can select a single project (one with or without sites), which will clear any other selected filters.

Clear Selected FiltersTo remove any selected filters, the user should re-select "Change filters", and click on the "Clear all" button in the bottom left.

 Exploring Filtered Projects and Sites
Exploring Filtered Projects and Sites

Using the Project ListOnce a filter has been applied, the results will appear in the "Project List" sidebar. Here the user can sort the list, hide projects without any sites, or download a ".csv" file containing the site information.

Site List OptionsBelow each project title is a full count of all the project's sites (left black toggle), and a count of the filtered sites (right orange toggle). The list of project sites can be expanded or collapsed by clicking on the small triangle to the left of the project title. The sites matching the filters appear in orange text, and those that don't match the filter appear in black text.

Highlighting a Site in the Project ListClicking on the name of a site in the Project List will highlight that site on the map. Sites within the project that do not match the filter show up as gray circles.

Highlighting Filtered Sites from the Project ListThe user can highlight all project sites that match the filters on the map by selecting the orange "Filtered Sites" toggle. Clicking on the project name will toggle the highlights off and on.

Highlighting All Sites from the Project ListThe user can highlight all project sites on the map by selecting the black "All Sites" toggle. Clicking on the project name will toggle the highlights off and on.
