User Guide

Search, Zoom
Search and Zoom Capabilities

Navigating the MapThe scroll wheel on the mouse can be used to zoom, and in the upper left area of the tool are "+" and "-" icons to zoom in and out, respectively.

Home LocationThe crosshair button will zoom the map to your location, and the home button will center the US on the map.

Opening a Site Information BoxTo open up a site information box, zoom in and hover over a site until the mouse hand is pointing at it, then click on the circle at its location on the map. This will open a popup window containing the site and project name.

Viewing Overlapping SitesWhen two or more sites are overlapping, their spots on the map appear darker.

Zoom into Overlapping SitesIn order to select overlapping sites, first zoom into them. When you are close enough to select them, a circle with a number inside will appear, representing the number of sites in that location.

Selecting Overlapping SitesOnce the circle containing the number of sites in the location is in view, click on that circle. The sites will separate, and the user can then click on each site to view their information.

Explanation

Explanation displayThe explanation box can be expanded or collapsed

Using the ExplanationThe expanded Explanation box contains a legend for the available sites and additional layers.

 Map Selection Options
Map Selection Options

Changing the Basemap StyleThe "Basemaps" panel allows a user to show various geographic information or landscape imagery as a background layer.

Changing or Adding LayersThe "Layers" sidebar allows a user to add or remove each individual layer.

Filtering Site and Project SelectionThe "Filters" sidebar allows a user to select sites or projects that match certain criteria.

Filter SitesOnce the user has selected "Change filters" on the "Filters" sidebar, the "Filter Sites" portion allows the user to select criteria to filter the site selection.

Filter SitesOnce the user has selected "Change filters" on the "Filters" sidebar, the "Filter Projects" portion allows the user to select criteria to filter the project selection.

Search by Project NameAlternatively, within the "Change filters" box, the user can filter sites by choosing the name of the project, which will clear any other selected filters.

 Data Display
Selection Capabilities and Site Information

Using the Project ListOnce a filter has been applied, information will appear in the "Project List" sidebar. Here the user can sort the list, hide any projects containing no sites, or download a ".csv" file containing the site information.

Viewing Project SitesThe list of project sites can be expanded or collapsed.

Showing All Sites Within a ProjectThe user can view the sites that meet the filtered criteria within a project (orange) by selecting "Filtered", or all sites within the project by selecting "All". Sites that do not meet the filtered criteria show up black.

Highlighting a Project in the Project ListThe user can highlight all of the sites within a project by clicking on its name in the "Project List" sidebar.

Highlighting a Site in the Project ListLikewise, clicking on the name of a site will highlight that site on the map.

Viewing Project InfoWhen a project is selected, you can view the project information using the project tabs beneath the map. Drag the top of the box to expand, and move between the tabs to view different project and site information.
